Proposal for O'Reilly Spark Summit San Francisco in June 2017

Track: Spark Ecosystem

Title: Exploring the Databricks Spark Reference Applications

Description: If you need to write a Spark application, why not start by copying a working program that demonstrates best practices, then modify it to suit your needs? The Spark Reference Applications are four such programs: 3 written in Scala and 1 in Java. Recently updated for Spark 2.0, this presentation discusses what these programs do, and goes into detail on some code that might otherwise be difficult to understand.

Abstract

This 30-minute presentation discusses:

  1. What are reference apps for, in general?
  2. Brief overview of the 4 Spark reference apps:
    1. Spark concepts presented.
    2. Scala programming style and functional programming idioms used.
    3. Java version of the Log Analyzer Reference App.
  3. Suggestions for how you might incorporate the concepts into your Spark project.
